- [x] 上一首、下一首
- [x] 删除歌曲
- [x] 选中列表中任一首歌功能

- [x] 播放器界面的设计，用户交互基础框架

- [x] 音乐播放、暂停、上一首、下一首、选中列表、快进、快退中任一首歌界面的实现（按钮美化）

- [x] 循环播放的实现
- [x] 歌词文件的读取和解析

| 功能          | 按键                 |
| :------------ | -------------------- |
| 播放/暂停     | /                    |
| 上一首/下一首 | Shift + </>          |
| 快进/快退     | Shift + Left / Right |
| 音量增大/减小 | Shift + Up / Down    |
| 关闭窗口      | Shift + F8           |
